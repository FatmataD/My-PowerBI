# My-PowerBI
# 📊 Power BI Data Analysis Project – Retail & Sales

## 📌 Overview
This project was completed as part of a Data Technician bootcamp and focuses on analyzing and visualizing retail and sales data using Power BI. The aim was to clean and transform raw data, build interactive reports, and communicate insights through effective data storytelling.

---

## 🛠️ Skills & Techniques Demonstrated

### 🔄 Data Transformation (Power Query)
- Cleaned and transformed raw datasets using **Power Query**  
- Handled missing values, corrected data types, and reshaped data for analysis  
- Combined and prepared multiple data sources for reporting  

---

### 📊 Report Development
- Built **interactive reports and dashboards** to present key business metrics  
- Structured layouts for clarity and ease of use  
- Enabled users to explore insights dynamically  

---

### 🎛️ Interactivity & Controls
- Implemented **slicers and filters** for dynamic data exploration  
- Allowed drill-down into specific dimensions such as region, product, and time  

---

### 🧮 DAX (Data Analysis Expressions)
- Created **calculated columns** for data enrichment  
- Developed **measures** to calculate KPIs such as total sales, averages, and growth metrics  
- Applied DAX functions to enhance analytical capabilities  

---

### 📈 Data Visualisation
Used a range of visualizations to effectively communicate insights:
- **Bar charts** – Comparing sales across categories  
- **Line charts** – Tracking trends over time  
- **Pie charts** – Showing proportions and distributions  
- **Maps** – Visualizing geographic performance  

---

## 📖 Data Storytelling
- Transformed complex data into **clear, actionable insights**  
- Designed reports to guide users through key findings  
- Highlighted trends, patterns, and performance drivers in retail data  

---

## 📈 Key Outcomes
- Delivered a comprehensive and interactive sales dashboard  
- Improved visibility into business performance and trends  
- Enabled data-driven decision-making through intuitive visuals  

---

## 🚀 Tools Used
- Power BI (Power Query, DAX, Interactive Reports)

---

## 📂 Project Focus
Retail and sales data analysis, including:
- Revenue tracking and performance analysis  
- Product and regional insights  
- Time-based trend analysis  

---

## 💡 What I Learned
- How to clean and transform data efficiently using Power Query  
- Writing effective DAX calculations for deeper analysis  
- Designing interactive and user-friendly dashboards  
- Communicating insights through strong data storytelling
